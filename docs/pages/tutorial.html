Tutorial

How to Use:

1.    Run the app. The system tray icon will appear in the bottom-right of the screen.

2.    Click on the system tray icon to activate the OCR screen (or use the keyboard shortcut "Ctrl+Alt" or "Ctrl+Windows").

3.    Select the text area on the computer screen (click at the top-left corner, drag to the bottom-right corner, and release the mouse).

4.    The text editor will pop up with editable text after the OCR conversion.

5.    Verify and edit the text using the built-in toolbar.

6.    Click "OK" button to finish (or click "Continue" button to proceed to the next selection).

7.    Paste the text into other text editors or appropriate applications as needed.

Keyboard Shortcuts:

•   Ctrl+Alt or Ctrl+Windows: Activate OCR screen

•   Right-click: Deactivate OCR screen

•   Double-click: Show Settings window

•   Ctrl+Q: Exit program

•   Esc: Close window

How to Show System Tray Icon:

1.    Press Windows+I to open the Settings window, and go to "Personalization" and "Taskbar".

2.    Under the "Notification area", click on "Select which icons appear on the taskbar", and turn on CubeOCR to show the System Tray Icon.
