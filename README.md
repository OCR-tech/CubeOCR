# CubeOCR
<a id="readme-top"></a>


**CubeOCR** is a tool designed to extract text from images and scanned documents.
- Simple & easy-to-use OCR tool for extracting any text on-screen
- Integrated built-in text editor with a transparent background
- Advanced image processing and noise filtering features
- Suitable for side-by-side paragraph and code editing
- Work offline for confidential & sensitive personal data
- Supports more than 107 different languages

## Description

**CubeOCR** project is a Python-based screenshot OCR tool. Simply select a text area, perform OCR, and ready to paste it anywhere.



### Features :
-  Fast and accurate text recognition
-  Instant and side-by-side integrated text editor  
-  Transparent background feature for built-in toolbar 
-  Easily copy & paste to other applications
-  Activate OCR screen using hotkeys
